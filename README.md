## Useful commands

- `npx cdk deploy` deploy this stack to your default AWS account/region
- `npx cdk diff` compare deployed stack with current state
- `npx cdk synth` emits the synthesized CloudFormation template
- `npm run cdk validate -- --unstable=validate` Checks ,CloudFormation template syntax, Resource configuration validity, IAM permissions, policies Asset publishing requirements
  `cdk diff` — Compare deployed vs. current state
  `cdk list` — List stacks in your app
  `cdk diagnose` — Diagnose CDK app for issues
